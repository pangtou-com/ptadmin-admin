# 列表指令 [@pt:cms::lists]
> 应用场景：栏目列表页
> 获取栏目下列表数据支持翻页
## 参数信息
| 参数名称 | 类型 | 是否必须 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| cid | int | 否 | 0 | 栏目ID，可设置多个栏目ID，多个栏目ID使用【|】分割，如：1|2|3 |
| fields | string | 否 | 空 | 指定读取字段，多个字段使用【|】分割，如：title|content |

## 调用示例

### 示例一
```php
    @pt:cms::lists(cid=123)
    
    @pt:end
```
